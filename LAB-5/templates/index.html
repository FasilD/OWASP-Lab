Welcome to the OWASP A05:2021 Security Misconfiguration Lab. This application has been intentionally configured with several common security flaws related to misconfiguration. Your goal is to identify and exploit these vulnerabilities, find the hidden flags, and then apply the necessary remediations to secure the application.

This lab is designed to provide hands-on experience for:

  • Understanding the impact of security misconfigurations.
  • Practicing common penetration testing techniques.
  • Learning how to remediate these vulnerabilities effectively.

Your Mission (as an Ethical Hacker)

Your task is to identify and exploit the following types of misconfigurations within this application. Look for "flags" which indicate a successful exploitation.

  • Default/Hardcoded Credentials
  • Verbose Error Messages
  • Sensitive File Exposure
  • Unnecessary Features / Default Content Exposure
  • Missing Security Headers
  • Overly Verbose Logging

**If you need guidance or hints to get started, please refer to the `testing-guide.md` file located in the lab's root directory.**
